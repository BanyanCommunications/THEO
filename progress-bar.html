JSON Generated Progress Bar

Documentation

0. Capabilities

  1. Clearly informs where you are in a course and how far you have to go
  2. JSON generated
  3. Can be used for any question type
  4. Each 'circle' is an <li> element and can easily be styled differently
  5. Can be styled horizontal or vertical
  6. Could be made clickable to allow users to go back

1. Dependencies

jQuery
JSON


2. JQuery Additional Functions

Two quick functions generate the progress bar.


function generateProgressBar(num){
	var progress = '';
	
	for( var j = 0; j < num; j++){
		var count = j + 1;
		progress += '<li class="progressItem"><span> </span></li>';
	}
	
	$('.progressBar').html(progress);
	updateProgress();
}

function updateProgress(){
	$('.progressBar li').removeClass('active');

}
