2. Cases

 case 'question':
 result = '<p class="incorrect">Wrong answer!</p>';
 break; 
 
 default: result = '<p class="correct">Correct answer!</p>';
 break;}

The 'case' is the class attached to the question that will be searched for when determining the correct message to display on submit. Currently these classes are in the HTML, but could be pulled in from a JSON (or similar) file to prevent easy cheating.

The 'result' is the message displayed on submit.


3. Validation

 $('.validate').append("<p>" + result + "</p>");

Appends 'result' variable message to the '.validate' div in the HTML. Currently messages are encoded in the Jquery, but could be stored and pulled in from a JSON file.
